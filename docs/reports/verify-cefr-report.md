> **Why this is deliberately narrow.** German unigram frequency is a weak CEFR grader: closed
> compounds are elementary yet individually rare ("der Geldautomat" is A2 but low-frequency), so
> **low frequency proves nothing**. The signal is reliable in one direction only: **high frequency
> is strong evidence a word is not advanced**. So the frequency tripwire flags just that case (a
> **common word carrying an advanced label**) and only for **vocabulary** (a collocation's level is
> phrase-level, not its noun's frequency). The unreliable direction is listed as information,
> never flagged. Recall is traded for precision so the flags are worth a human's time. A real CEFR
> verdict is the AI jury (Layer 4) + human audit (Layer 5). Ladder: A2 · B1.1 · B1.2 · B2.1 · B2.2 · C1.

## Method & caveats

- **Frequency source:** `wordfreq` German Zipf (vendored `scripts/vendor/german-frequency-subset.json`,
  built by `pnpm build:frequency-subset`). Runs fully offline, like the Phase A oracles.
- **Bands:** Zipf ≥5 → A2, ≥4.4 → B1.1, ≥3.8 → B1.2, ≥3.2 → B2.1, ≥2.6 → B2.2, else C1. The edges
  only need to isolate *genuinely common* words; the rest is intentionally coarse.
- **Precision over recall, one reliable direction, vocabulary only.** This is a *tripwire* pointing
  the jury/human at the grossest, most defensible mislabels, not a CEFR grader and never a gate.
