## Method & caveats

- **Tool:** LanguageTool 6.8 `language-de`, resolved from Maven Central and run fully offline
  (`scripts/build-languagetool.mjs`). The engine is ~69 MB, too large to vendor, so it is cached
  from an allowed host rather than committed (the Phase A pattern, at a size that forbids vendoring).
- **Warn-only:** LanguageTool over-flags idiomatic and domain German, so no build gate depends on
  this. It is a triage list for a human/jury pass (ladder Layers 4–5), not a pass/fail.
- **Ignore-list:** `WHITESPACE_RULE`, `DE_DATE_WEEKDAY_CURRENTYEAR` (rules reviewed as pure noise for curated copy).
- A clean sentence here is checked for grammar/spelling only, not for naturalness, register, or
  CEFR level (higher rungs of the ladder: the CEFR heuristic in `verify:cefr`, then the AI jury).
